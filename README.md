# RROML

---

**Short Description:**

RROML is a mod loader for the game Railroads Online! it makes it so that the community, can make mods for the game to imrpove QoL, add new things, and much more!

---

**Information:**
- RROML is coded in C++ & C#
- Mods are coded in C#

---

**Installation:**
